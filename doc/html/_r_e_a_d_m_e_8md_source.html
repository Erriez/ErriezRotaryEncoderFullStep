1 # 3 speed Rotary Encoder Full Step library for Arduino
2 
3 This is an optimized three speed Rotary Encoder library for Arduino which supports:
4 
5 * Full step Rotary Encoder types.
6 * Detect three rotation speeds.
7 * Configurable rotation speed sensitivity.
8 * Polling and interrupt based.

15 ## Full step / half step Rotary Encoders
16 
17 The difference between a full step or half step Rotary Encoder type is how the data signals of the two pins are generated. It depends on the mechanical construction of the notches and contacts inside the Rotary Encoder.
18 
19 Please refer to the [ErriezRotaryEncoderHalfStep](https://github.com/Erriez/ErriezRotaryEncoderHalfStep) library for half step Rotary Encoders.
20 Experiment with the half step and full step libraries which works optimal for your Rotary Encoder.

42 ### Interrupts
43 
44 Both rotary pins must be connected to a DIGITAL pin with interrupt support, such as `INT0` or `INT1`. This is chip specific. Please refer to the documentation of your board or [attachInterrupt()](https://www.arduino.cc/reference/en/language/functions/external-interrupts/attachinterrupt/).

100 ## Usage
101 
102 **Read rotary with polling**
103 ```c++
104 #include <RotaryFullStep.h>
105 
106 // Connect rotary pins to the DIGITAL pins of the Arduino board
107 #define ROTARY_PIN1 2
108 #define ROTARY_PIN2 3
109 
110 // Enable ONE of the three constructors below with different number of arguments:
111 
112 // Initialize full step rotary encoder, default pull-up enabled, default
113 // sensitive=100
114 RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2);
115 
116 // Or initialize full step rotary encoder, pull-up disabled, default sensitive=100
117 // RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2, false);
118 
119 // Or initialize full step rotary encoder, pull-up enabled, sensitive 1..255
120 // A higher value is more sensitive
121 // RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2, true, 150);
122 
123 void loop()
124 {
125  int rotaryState = rotary.read();
126 
127  // rotaryState = -3: Counter clockwise turn, multiple notches fast
128  // rotaryState = -2: Counter clockwise turn, multiple notches
129  // rotaryState = -1: Counter clockwise turn, single notch
130  // rotaryState = 0: No change
131  // rotaryState = 1: Clockwise turn, single notch
132  // rotaryState = 2: Clockwise turn, multiple notches
133  // rotaryState = 3: Clockwise turn, multiple notches fast
134 }
135 ```
136 
137 **Read rotary with interrupts**
138 
139 ```c++
140 #include <RotaryFullStep.h>
141 
142 // Connect rotary pins to Arduino DIGITAL pins with interrupt support:
143 //
144 // +-----------------------------------+--------------------------+
145 // | Board | DIGITAL interrupt pins |
146 // +-----------------------------------+--------------------------+
147 // | Uno, Nano, Mini, other 328-based | 2, 3 |
148 // | Mega, Mega2560, MegaADK | 2, 3, 18, 19, 20, 21 |
149 // | Micro, Leonardo, other 32u4-based | 0, 1, 2, 3, 7 |
150 // +-----------------------------------+--------------------------+
151 //
152 #define ROTARY_PIN1 2
153 #define ROTARY_PIN2 3
154 
155 // Enable ONE of the three constructors below with different number of arguments:
156 
157 // Initialize full step rotary encoder, default pull-up enabled, default
158 // sensitive=100
159 RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2);
160 
161 // Or initialize full step rotary encoder, pull-up disabled, default sensitive=100
162 // RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2, false);
163 
164 // Or initialize full step rotary encoder, pull-up enabled, sensitive 1..255
165 // A higher value is more sensitive
166 // RotaryFullStep rotary(ROTARY_PIN1, ROTARY_PIN2, true, 150);
167 
168 void setup()
169 {
170  // Initialize pin change interrupt on both rotary encoder pins
171  attachInterrupt(digitalPinToInterrupt(ROTARY_PIN1), rotaryInterrupt, CHANGE);
172  attachInterrupt(digitalPinToInterrupt(ROTARY_PIN2), rotaryInterrupt, CHANGE);
173 }
174 
175 void rotaryInterrupt()
176 {
177  int rotaryState = rotary.read();
178 
179  // rotaryState = -3: Counter clockwise turn, multiple notches fast
180  // rotaryState = -2: Counter clockwise turn, multiple notches
181  // rotaryState = -1: Counter clockwise turn, single notch
182  // rotaryState = 0: No change
183  // rotaryState = 1: Clockwise turn, single notch
184  // rotaryState = 2: Clockwise turn, multiple notches
185  // rotaryState = 3: Clockwise turn, multiple notches fast
186 }
187 ```

191 ## Libary installation using Git
192 
193 `Git` is the preferred way to keep this library up to date, because the Arduino Library manager does not update as long as this library is not added to the official Arduino Library database.
194 
195 ### Install Git client for Windows
196 
197 Install a [Git client for Windows](https://git-scm.com/download/win).

201 Open a command prompt and install a Git client for Linux, such as Debian Ubuntu:
202 
203 ```shell

205 ```

207 ### Get Arduino libraries directory
208 
209 This library must be installed in the Arduino Sketchbook library subdirectory.
210 
211 To retrieve the Arduino Sketchbook directory, open the Arduino IDE Preferences dialog box via:
212 `File` | `Preferences` |`Settings tab` and copy the Sketchbook location.
213 
214 For example on:
215 
216 * Windows : `C:\Users\User\Documents\Arduino`
217 * Linux: `/home/user/Arduino`
218 
219 ### Clone this library
220 
221 Clone this library by opening a command prompt:
222 
223 * Windows: (`Windows key + R`, Type `cmd` + `[ENTER]`)
224 * Linux: Depends on your version.
225 
226 Then type:
227 
228 ```shell
229 # Change directory to the sketchbook directory as configured in the Arduino IDE:
230 # Windows:
231 cd C:\Users\User\Documents\Arduino
232 # Linux:
233 cd ~/Arduino
234 
235 # Go to the libraries subdirectory
236 cd libraries
237 
238 # Run the git clone library once:
239 git clone https://github.com/Erriez/ErriezRotaryEncoderFullStep.git
240 ```
